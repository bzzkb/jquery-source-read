初始化：
git init：初始化一个git版本库
git remote add master https://github.com/jacksparrow01/jquery-source-read.git
git config --global user.email ""
git config --global user.name ""

操作过程：

git add：将文件加入到版本库中

git commit -a -m ''：将修改的文件的内容提交到版本库中

git push master master	将本地的master提交到远程的master中

git pull master	master	相当于svn update


git clone git_url	将远程版本库拷贝到本地
